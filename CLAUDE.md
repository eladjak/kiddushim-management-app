# Kidushishi Management - הנחיות פיתוח

## הפרויקט
אפליקציית ניהול בטייפסקריפט - ממשק משתמש, ניהול נתונים, ודוחות.

## עקרונות
- UX ידידותי ונגיש
- ביצועים מהירים
- קוד מתוחזק
- בדיקות מקיפות

## טכנולוגיות
- TypeScript
- React

## פקודות
- `bun run build` - בנייה
- `bun test` - בדיקות
- `bunx tsc --noEmit` - בדיקת טייפים
