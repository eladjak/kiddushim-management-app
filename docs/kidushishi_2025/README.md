# פרויקט קידושישי 2025 📚

## מטרת הפרויקט

פרויקט "קידושישי 2025" הוא יוזמה קהילתית ליצירת מסורת של קבלות שבת במגדל העמק, המחברת בין כל חלקי האוכלוסייה דרך חוויה משותפת של קדושת השבת, מוזיקה, לימוד ויצירה.

המסמכים בתיקייה זו מהווים את התיעוד המפורט של הפרויקט, כולל תכנית עבודה, לוח זמנים, תהליכי עבודה, תבניות הפעלה, ואפיון מערכת מידע ייעודית.

## אינטגרציה במערכת הקיימת

פרויקט קידושישי 2025 ישולב בתוך מערכת הניהול הכללית של קידושישי שנמצאת בתהליך שדרוג.
המערכת המשודרגת תכלול מודולים ייעודיים לניהול פרויקט זה, שיפותחו לפי האפיון המפורט.

## הרשאות ובעלי תפקידים

הפרויקט כולל מספר בעלי תפקידים, כל אחד עם הרשאות וגישה מותאמת למערכת:
- רכז ראשי
- רכזי משנה
- בנות שירות
- מתנדבי נוער
- שותפים חיצוניים

## לוחות זמנים

- **שלב הכנה**: שבט-אדר 2025
- **שלב יח"צ וגיוס**: אדר-ניסן 2025
- **שלב הפעלה**: אייר 2025 ואילך
- **סיכום שנתי**: חשוון 2026

## צעדים מיידיים

1. אישור והטמעת תכנית הפעולה
2. בניית מערכת ניהול על בסיס האפיון
3. הקמת צוות מוביל
4. יצירת תוכנית תקציבית מפורטת יותר
5. התחלת פעילות בבתי ספר
