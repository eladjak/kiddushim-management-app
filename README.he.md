# קידושישי - מערכת ניהול לארגון

מערכת ניהול מקיפה לארגון קידושישי, הכוללת ניהול חברים, אירועים, תשלומים ותקשורת.

## תכונות

- ניהול חברים ומנויים
- ניהול אירועים וקידושים
- מעקב תשלומים
- דוחות ותובנות
- ממשק ניהול נוח

## טכנולוגיות

- React
- Node.js
- MongoDB

## התקנה

```bash
git clone https://github.com/eladjak/kidushishi-menegment-app.git
cd kidushishi-menegment-app
npm install
npm start
```
